# Features

## Transaction Management
- Import CSV and PDF bank statements
- Automatic categorization via ML agents
- Deduplication of repeated transactions
- Edit, delete, and review transactions
- Search & filter by date, amount, category, direction, and labels

## Dashboard
- Bento-grid layout with glassmorphism design
- Balance overview, income/expense summary
- Monthly bar chart
- Category spending with progress bars
- Quick actions grid
- Staggered animations and shimmer loading

## Budget Tracking
- Set monthly budgets per category
- Visual progress bars showing spending
- Warning at 80% threshold
- Alert when exceeded
- Supports multiple categories independently

## Credit Cards
- Track cards with name, issuer, last four digits
- Credit limit and available credit monitoring
- Automatic `outstandingAmount` and `isOverLimit` computation
- Link transactions to credit cards

## Accounts
- Multi-type accounts: Savings, Credit Card, Wallet, Cash, Investment
- Track balance, bank name, currency
- Active/inactive status

## Labels
- Create colored labels for transaction tagging
- 16-preset color picker
- System labels vs user labels
- Multi-label support per transaction

## Recurring Payment Detection
- Automatic detection of monthly/weekly recurring payments
- Frequency analysis with variance calculation
- Identifies patterns in transaction history

## Insights
- Spending patterns and trends
- Category breakdowns
- Agent action logs for transparency

## Security
- Optional passcode lock
- Biometric authentication (Face ID / Touch ID)
- Privacy guard for sensitive data
- All data stored locally — no cloud upload

## Data Management
- Import history with batch tracking
- Export transactions
- Review queue for flagged items
- Clear all data option

## Theme
- Premium dark theme with glassmorphism
- Froster glass cards with backdrop blur
- Custom color palette (purple/teal accents)
- Staggered list animations
- Page transitions (slide + fade)
